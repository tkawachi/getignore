getignore

It fetches .gitignore from Github API. Useful when creating new git repositories.

Install

$ curl -O https://raw.github.com/tkawachi/getignore/master/getignore
$ chmod +x getignore

Usage

# To fetch .gitignore for Scala
$ getignore scala
